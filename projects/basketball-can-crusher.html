Idea / Concept

Automatic Basketball Can Crusher

A planned arcade-style pneumatic can crusher that uses a basketball-inspired loading system, automated sensing, controlled crushing, and mechanical ejection.

← Back to Portfolio

Project Overview

The Automatic Basketball Can Crusher is a concept for a wall-mounted machine that turns recycling cans into an interactive mechanical process. Instead of simply placing a can into a normal crusher, the user would drop or shoot the can into a basketball-style intake system.

The can would travel down a gravity-fed track, enter a chute, trigger a sensor, and then be crushed automatically using a pneumatic cylinder. After crushing, the machine would eject the can into a collection area.

Design Concept

Basket and Gravity Feed

The machine is planned to have a raised basket or intake area separated from the main crushing mechanism. A wireformed track would guide the can from the basket into the machine using gravity, similar to an arcade or pinball-style mechanism.

Chute and Sensor

The can would enter a chute where a microswitch or similar sensor detects that a can has passed into position. This sensor would trigger the automated crushing sequence.

Pneumatic Crushing

The crushing motion would be powered by compressed air. A pneumatic cylinder would drive a crushing plate into the can, flattening it inside a protected chamber.

Automatic Ejection

After the can is crushed, a secondary mechanism would eject the flattened can from the crushing chamber. The current design concept includes a spring-assisted or pneumatic horizontal ejection system.

Safety Features

  • Covered crushing chamber to prevent finger access
  • Spring-assisted protection near the chute entrance
  • Physical separation between the basket and crushing mechanism
  • Plexiglass or mesh viewing area to show motion while keeping the user protected
  • Controlled triggering using a microswitch or sensor

Planned System Components

  • Wall-mounted frame or panel
  • Basket-style can intake
  • Wireformed gravity track
  • Main chute and sensor
  • Pneumatic crushing cylinder
  • Small compressed air tank mounted on the machine
  • Pressure gauge and air hose connector
  • Arduino or similar controller
  • Speaker for arcade-style sound effects
  • Back-mounted electronics area with spacing from the wall

Engineering Goals

  • Design a mechanically reliable can loading and crushing sequence
  • Use pneumatics to create fast and repeatable motion
  • Integrate sensors, controls, and mechanical safety features
  • Create a visually interesting machine with an arcade-style feel
  • Practice fabrication, mounting, wiring, and system integration

Current Status

This project is currently in the idea and concept phase. The main mechanical process, safety approach, pneumatic system concept, and user interaction have been planned, but physical fabrication has not started yet.
